# ✔️About
<br>

**기초가 튼튼하고 실력있는 프론트엔드 개발자로 성장하기 위해 자바스크립트의 기본 개념과 동작 원리를 잘 숙지하는 것이 개인적으로 중요하다고 생각합니다. 책을 읽고 공부한 내용을 복습하기 위해 간략히 정리한 마인드맵으로 구성된 리포지토리입니다.**
